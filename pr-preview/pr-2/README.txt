Como publicar rapidamente:
1) Netlify: arraste o ZIP para o painel do Netlify.
2) GitHub Pages: crie um repositório, envie os arquivos e habilite Pages (branch main, /root).
3) Vercel: importe o repositório (ou arraste a pasta) e faça o deploy estático.
